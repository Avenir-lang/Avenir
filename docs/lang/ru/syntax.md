# Синтаксис

Этот документ описывает синтаксис языка Avenir.

## Структура программы

Каждая программа начинается с объявления пакета:

```avenir
pckg <name>;
```

После этого могут идти импорты, структуры, интерфейсы и функции.

## Ключевые слова

Зарезервированные слова:

- `pckg`, `import`, `fun`, `struct`, `interface`
- `pub`, `mut`, `var`
- `if`, `else`, `while`, `for`, `break`, `return`
- `try`, `catch`, `throw`
- `async`, `await`
- `true`, `false`, `none`, `some`

## Ключевые слова типов

- `int`, `float`, `string`, `bool`, `bytes`
- `void`, `any`, `error`
- `list`, `dict`

## Операторы

- арифметические: `+`, `-`, `*`, `/`, `%`
- сравнения: `==`, `!=`, `<`, `<=`, `>`, `>=`
- логические: `&&`, `||`, `!`
- присваивание: `=`

## Символы

- `|` для аннотации типов (`name | type`)
- `;`, `,`, `.`, `:`, `?`
- `()`, `{}`, `[]`, `<>`

## Комментарии

Поддерживаются однострочные и многострочные комментарии.

```avenir
// comment
/* multi-line
   comment */
```

## Литералы

Поддерживаются литералы:

- целые и float
- строки и bytes
- bool (`true`, `false`)
- list (`[1, 2]`) и dict (`{ key: value }`)
- struct-литералы (`Point{x = 1, y = 2}`)

## Приоритет выражений

От большего к меньшему: primary → unary → multiplicative → additive →
relational → equality → logical AND → logical OR.

## Терминаторы

Операторы завершаются `;`, блоки задаются `{ ... }`.
