# Функции

Функции в Avenir являются полноправными значениями. Они могут быть именованными
или анонимными (литералами функций).

## Объявление функции

Функции объявляются с помощью `fun`:

```avenir
fun name(param1 | type1, param2 | type2) | returnType {
    
}
```

Возвращаемый тип обязателен. Для отсутствия результата используйте `void`.

### Асинхронные функции

Функции можно объявлять как асинхронные:

```avenir
async fun fetchNumber() | int {
    return 10;
}
```

Вызов async-функции возвращает значение `Future`.

### `await`

`await` используется внутри async-функций для получения результата:

```avenir
async fun main() | int {
    var a | int = await fetchNumber();
    return a + 1;
}
```

### Дженерики

Функции могут иметь параметры типов:

```avenir
fun identity<T>(x | T) | T {
    return x;
}
```

Аргументы типов в вызове указываются явно:

```avenir
var n | int = identity<int>(42);
```

Вывод аргументов обобщённых типов пока не поддерживается.

## Публичные функции

`pub fun` делает функцию доступной из других модулей.

## Параметры

Параметры всегда имеют явные аннотации типов.

### Значения по умолчанию

Параметры могут иметь значения по умолчанию, известные во время компиляции.

### Именованные аргументы

Поддерживаются именованные аргументы:

```avenir
create_point(x = 10, y = 20);
```

Позиционные аргументы должны идти раньше именованных.

## Возврат значений

Возврат выполняется через `return`. Для `void` можно использовать `return;` без значения.

## Литералы функций и замыкания

Анонимные функции поддерживаются и могут захватывать переменные внешней области
видимости.

## Типы функций

Тип функции записывается как:

```avenir
fun (int, int) | int
```

## Вызовы функций

Функции вызываются через круглые скобки, методы — через точечную нотацию.

## `main`

Каждая программа должна иметь точку входа `main`.

```avenir
fun main() | void {
}
```

`main` также может быть асинхронной.
