# Дорожная карта

Этот документ описывает возможные будущие улучшения. По мере развития языка
пункты могут меняться.

## Ядро языка

- Нативный бэкенд для `avenir build -target=native` (в настоящее время не реализован)
- Вывод аргументов обобщённых типов
- Улучшение удобства обобщённых коллекций
- Опциональная цепочка (`x?.field`) и связанная с этим эргономика
- Сопоставление с образцом / `match`-выражения
- Оператор `continue` в циклах

## Среда выполнения и VM

- Улучшенная диагностика (стек-трейсы, метаданные ошибок)
- Хуки для профилирования производительности

## Стандартная библиотека

- Примитивы асинхронного ввода-вывода
- Расширенные файловые API (метаданные, итерация директорий)
- Улучшения HTTP (TLS, middleware, потоковые тела)
- Поддержка WebSocket

## Инструменты

- Форматтер и линтер
- Менеджер пакетов / реестр
