# Переменные

Переменные в Avenir должны объявляться явно с указанием типа.

## Объявление переменной

Переменные объявляются с помощью `var`:

```avenir
var name | type = value;
```

Аннотация типа обязательна. Начальное значение тоже обязательно.

Пример:

```avenir
var x | int = 10;
var name | string = "Avenir";
var is_ready | bool = true;
```

## Присваивание

Переменным можно присваивать новые значения через оператор `=`:

```avenir
var x | int = 10;
x = 20;
```

Присваиваемое значение должно быть совместимо с типом переменной.

## Область видимости

Переменные видны в пределах блока, где они объявлены:

```avenir
{
    var x | int = 10;
    // x is available here
}
// x is not available here
```

Параметры функции видны внутри тела функции.

## Видимость переменных

Переменные всегда приватны для своей области видимости. Экспортировать переменную из модуля нельзя.

## Требования к типам

Все переменные должны иметь явную аннотацию типа. Вывод типов не поддерживается.

Пример:

```avenir
var x | int = 10;
var y = 10;
```
