# Операторы

Этот документ описывает операторы Avenir и правила их типизации.

## `+` (сложение и конкатенация строк)

Оператор `+` разрешается type checker-ом и поддерживает ровно две формы:

- числовое сложение: `int + int`, `float + float` или смешанное `int`/`float`
- конкатенация строк: `string + string`

Неявные преобразования не выполняются. В частности:

- `string + int` — ошибка компиляции
- `int + string` — ошибка компиляции

Примеры:

```avenir
var a | string = "hello " + "world";
var n | int = 1 + 2;
var f | float = 1 + 2.5;
```

Если нужно встроить нестроковые значения, используйте интерполяцию строк:

```avenir
print("sum = ${1 + 2}");
```
