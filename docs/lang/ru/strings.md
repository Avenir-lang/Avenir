# Строки

Avenir поддерживает строковые литералы как в двойных кавычках (`"..."`), так и
в одинарных (`'...'`). Обе формы эквивалентны и создают значение `string`.

Строки в одинарных кавычках **не** являются символами. Типа `char` нет.

Примеры:

```avenir
var a | string = "hello";
var b | string = 'world';
var c | string = a + " " + b;
```

## Экранирование

Оба вида кавычек поддерживают одинаковые escape-последовательности:

- `\\` обратный слеш
- `\"` двойная кавычка
- `\'` одинарная кавычка
- `\n` перевод строки
- `\t` табуляция
- `\r` возврат каретки
- `\0` нулевой байт
- `\uXXXX` кодовая точка Unicode в шестнадцатеричной форме
- `\xNN` байт в шестнадцатеричной форме

Примеры:

```avenir
var s1 | string = "line\nbreak";
var s2 | string = 'quote: \'';
var s3 | string = "backslash: \\";
```

Строки не могут содержать необработанный перевод строки. Используйте `\n`.

## Интерполяция

Интерполяция строк использует `${ ... }` и работает одинаково в обоих стилях кавычек:

```avenir
var name | string = 'Avenir';
print("hello ${name}");
print('hello ${name}');
```

Интерполяция преобразует встроенные значения в строки так же, как это делает `print`.

## Конкатенация

Оператор `+` поддерживает конкатенацию строк только когда **оба операнда —
строки**. Неявных преобразований для `+` нет. Для включения нестроковых значений
используйте интерполяцию.
