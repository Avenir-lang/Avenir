# Обзор языка Avenir

Avenir — статически типизированный язык программирования для создания
надежного ПО. Язык сочетает читаемый синтаксис, строгую типизацию и
полноценную среду выполнения.

## Ключевые возможности

- строгая статическая типизация без неявного вывода
- union- и опциональные типы (`<A|B>`, `T?`)
- структуры, интерфейсы, методы, замыкания
- дженерики с явными аргументами типов
- модули и импорты с алиасами
- async/await и выполнение на VM

## Основные типы

- примитивы: `int`, `float`, `string`, `bool`, `bytes`
- коллекции: `list<T>`, `dict<T>`
- пользовательские типы: `struct`, `interface`
- функции как полноправные значения
- встроенный тип `error`

## Философия

Avenir ориентирован на:

- безопасность типов
- явность и читаемость кода
- практичность для прикладной разработки
- эффективное исполнение байткода в VM

## Пример

```avenir
pckg main;

struct Point {
    x | int
    y | int
}

fun (p | Point).sum() | int {
    return p.x + p.y;
}

fun main() | void {
    var p | Point = Point{x = 3, y = 4};
    print(p.sum());
}
```
