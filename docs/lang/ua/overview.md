# Огляд мови Avenir

Avenir — статично типізована мова програмування для створення
надійного ПЗ. Мова поєднує читабельний синтаксис, строгу типізацію і
повноцінне середовище виконання.

## Ключові можливості

- строга статична типізація з локальним виведенням типів
- union- і опціональні типи (`<A|B>`, `T?`)
- структури, інтерфейси, методи, замикання
- дженеріки з явними аргументами типів
- модулі та імпорти з аліасами
- async/await і виконання у VM

## Основні типи

- примітиви: `int`, `float`, `string`, `bool`, `bytes`
- колекції: `list<T>`, `dict<K, V>` (або `dict<V>`)
- користувацькі типи: `struct`, `interface`
- функції як повноправні значення
- вбудований тип `error`

## Філософія

Avenir орієнтована на:

- безпеку типів
- явність і читабельність коду
- практичність для прикладної розробки
- ефективне виконання байткоду у VM

## Приклад

```avenir
pckg main;

struct Point {
    x | int
    y | int
}

fun (p | Point).sum() | int {
    return p.x + p.y;
}

fun main() | void {
    var p | Point = Point{x = 3, y = 4};
    print(p.sum());
}
```
