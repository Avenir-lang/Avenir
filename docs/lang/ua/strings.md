# Рядки

Avenir підтримує рядкові літерали як у подвійних лапках (`"..."`), так і в
одинарних (`'...'`). Обидві форми еквівалентні й створюють значення `string`.

Рядки в одинарних лапках **не** є символами. Типу `char` немає.

Приклади:

```avenir
var a | string = "hello";
var b | string = 'world';
var c | string = a + " " + b;
```

## Екранування

Обидва види лапок підтримують однакові escape-послідовності:

- `\\` зворотний слеш
- `\"` подвійна лапка
- `\'` одинарна лапка
- `\n` новий рядок
- `\t` табуляція
- `\r` повернення каретки
- `\0` нульовий байт
- `\uXXXX` кодова точка Unicode у шістнадцятковій формі
- `\xNN` байт у шістнадцятковій формі

Приклади:

```avenir
var s1 | string = "line\nbreak";
var s2 | string = 'quote: \'';
var s3 | string = "backslash: \\";
```

Рядки не можуть містити необроблений новий рядок. Використовуйте `\n`.

## Інтерполяція

Інтерполяція рядків використовує `${ ... }` і працює однаково в обох стилях лапок:

```avenir
var name | string = 'Avenir';
print("hello ${name}");
print('hello ${name}');
```

Інтерполяція перетворює вбудовані значення на рядки так само, як це робить `print`.

## Конкатенація

Оператор `+` підтримує конкатенацію рядків лише коли **обидва операнди —
рядки**. Неявних перетворень для `+` немає. Для включення не-рядкових значень
використовуйте інтерполяцію.
