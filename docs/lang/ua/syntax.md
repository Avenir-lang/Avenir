# Синтаксис

Цей документ описує синтаксис мови програмування Avenir.

## Структура програми

Кожна програма Avenir починається з оголошення пакета:

```avenir
pckg <name>;
```

Ім'я пакета може бути простим ідентифікатором або шляхом через крапку (наприклад, `std.net`, `app.utils`).

Після оголошення пакета програма може містити:

1. **Імпорти** (опціонально, нуль або більше)
2. **Оголошення структур** (опціонально, нуль або більше)
3. **Оголошення функцій** (принаймні одна, включно з `main`)

Структури та функції можуть оголошувати узагальнені параметри типів в кутових дужках після імені (наприклад, `struct Box<T> { ... }`, `fun id<T>(x | T) | T { ... }`).

## Ключові слова

Зарезервовані слова:

- `pckg` — оголошення пакета
- `import` — оголошення імпорту
- `fun` — оголошення функції
- `struct` — оголошення структури
- `pub` — модифікатор видимості
- `mut` — модифікатор мутабельності
- `var` — оголошення змінної
- `if`, `else` — умовні оператори
- `while` — цикл while
- `for` — цикл for
- `return` — оператор повернення
- `throw` — генерація виключення
- `try`, `catch` — обробка виключень
- `break` — вихід з циклу
- `true`, `false` — булеві літерали
- `none`, `some` — літерали optional

## Ключові слова типів

- `int` — цілочисельний тип
- `float` — тип з плаваючою комою
- `string` — рядковий тип
- `bool` — булевий тип
- `void` — тип void (немає значення повернення)
- `any` — тип any
- `error` — тип помилки
- `bytes` — тип байтів
- `list` — тип списку

## Оператори

### Арифметичні

- `+` — додавання
- `-` — віднімання
- `*` — множення
- `/` — ділення
- `%` — остача від ділення (тільки цілі числа)

### Порівняння

- `==` — рівність
- `!=` — нерівність
- `<` — менше
- `<=` — менше або дорівнює
- `>` — більше
- `>=` — більше або дорівнює

### Логічні

- `&&` — логічне І
- `||` — логічне АБО
- `!` — логічне НЕ

### Присвоєння

- `=` — присвоєння

## Символи

- `|` — роздільник типу (використовується в `name | type`)
- `;` — термінатор оператора
- `,` — роздільник списку
- `.` — доступ до члена
- `(` `)` — дужки (групування, виклики функцій)
- `{` `}` — фігурні дужки (блоки, літерали структур)
- `[` `]` — квадратні дужки (літерали списків, індексація)
- `<` `>` — кутові дужки (параметри типів, union-типи)
- `?` — суфікс опціонального типу
- `:` — роздільник ключа словника (в dict-літералах: `key: value`)

## Коментарі

Avenir підтримує однорядкові та багаторядкові коментарі:

```avenir
// Однорядковий коментар

/*
 * Багаторядковий коментар
 * Може займати кілька рядків
 */
```

## Ідентифікатори

Ідентифікатори починаються з літери або підкреслення, далі йдуть літери, цифри або підкреслення.

## Літерали

### Цілочисельні літерали

```avenir
42
-10
0
```

### Літерали з плаваючою комою

```avenir
3.14
-0.5
1.0
```

### Рядкові літерали

```avenir

```

### Булеві літерали


### Літерали байтів

```avenir

```

### Літерали списків


### Літерали словників


### Літерали структур

## Пріоритет виразів

Від найвищого до найнижчого:

1. Первинні вирази (літерали, ідентифікатори, дужки)
2. Унарні оператори (`!`, `-`)
3. Мультиплікативні (`*`, `/`, `%`)
4. Адитивні (`+`, `-`)
5. Реляційні (`<`, `<=`, `>`, `>=`)
6. Рівності (`==`, `!=`)
7. Логічне І (`&&`)
8. Логічне АБО (`||`)

## Термінатори операторів

Оператори завершуються крапкою з комою (`;`):

```avenir
var x | int = 10;
x = 20;
print(x);
```

Блоки використовують фігурні дужки і не потребують крапки з комою:

```avenir
{
    var x | int = 10;
    print(x);
}
```
