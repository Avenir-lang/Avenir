# Синтаксис

Цей документ описує синтаксис мови Avenir.

## Структура програми

Кожна програма починається з оголошення пакета:

```avenir
pckg <name>;
```

Після цього можуть іти імпорти, структури, інтерфейси та функції.

## Ключові слова

Зарезервовані слова:

- `pckg`, `import`, `fun`, `struct`, `interface`
- `pub`, `mut`, `var`
- `if`, `else`, `while`, `for`, `break`, `return`
- `try`, `catch`, `throw`
- `async`, `await`
- `true`, `false`, `none`, `some`

## Ключові слова типів

- `int`, `float`, `string`, `bool`, `bytes`
- `void`, `any`, `error`
- `list`, `dict`

## Оператори

- арифметичні: `+`, `-`, `*`, `/`, `%`
- порівняння: `==`, `!=`, `<`, `<=`, `>`, `>=`
- логічні: `&&`, `||`, `!`
- присвоєння: `=`

## Символи

- `|` для анотації типів (`name | type`)
- `;`, `,`, `.`, `:`, `?`
- `()`, `{}`, `[]`, `<>`

## Коментарі

Підтримуються однорядкові та багаторядкові коментарі.

```avenir
// comment
/* multi-line
   comment */
```

## Літерали

Підтримуються літерали:

- цілі та float
- рядки і bytes
- bool (`true`, `false`)
- list (`[1, 2]`) і dict (`{ key: value }`)
- struct-літерали (`Point{x = 1, y = 2}`)

## Пріоритет виразів

Від більшого до меншого: primary → unary → multiplicative → additive →
relational → equality → logical AND → logical OR.

## Термінатори

Оператори завершуються `;`, блоки задаються `{ ... }`.
