# Дорожня карта

Цей документ описує можливі майбутні покращення. У міру розвитку мови пункти
можуть змінюватися.

## Ядро мови

- Нативний бекенд для `avenir build -target=native` (наразі не реалізовано)
- Виведення аргументів узагальнених типів
- Покращення зручності узагальнених колекцій
- Опціональний ланцюжок (`x?.field`) і пов'язані з цим покращення
- Зіставлення зі зразком / `match`-вирази
- Оператор `continue` у циклах

## Середовище виконання і VM

- Покращена діагностика (стек-трейси, метадані помилок)
- Хуки для профілювання продуктивності

## Стандартна бібліотека

- Примітиви асинхронного введення-виведення
- Розширені файлові API (метадані, ітерація директорій)
- Покращення HTTP (TLS, middleware, потокові тіла)
- Підтримка WebSocket

## Інструменти

- Форматер і лінтер
- Менеджер пакунків / реєстр
