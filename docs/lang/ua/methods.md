# Методи

Методи — це функції, пов'язані з типом. Avenir підтримує методи екземпляра та статичні методи.

## Методи екземпляра

Метод екземпляра — це функція з параметром-отримувачем.

### Оголошення

Методи екземпляра оголошуються через синтаксис отримувача:

```avenir
fun (self | Point).sum() | int {
    return self.x + self.y;
}
```

Синтаксис отримувача: `(name | Type).methodName`.

### Виклик методів екземпляра

Методи викликаються через крапкову нотацію:

```avenir
var p | Point = Point{x = 3, y = 4};
var s | int = p.sum();
```

### Доступ до приватних полів

Методи екземпляра можуть звертатися до приватних полів отримувача.

```avenir
struct Point {
    x | int
    y | int
}

fun (self | Point).sum() | int {
    return self.x + self.y;
}
```

### Змінювальні методи

Методи екземпляра можуть змінювати змінювані поля.

```avenir
mut struct Point {
    x | int
    y | int
}

fun (self | Point).move(dx | int, dy | int) | void {
    self.x = self.x + dx;
    self.y = self.y + dy;
}
```

## Статичні методи

Статичні методи пов'язані з типом, але не мають екземпляра-отримувача.

### Оголошення

Статичні методи оголошуються через ім'я типу:

```avenir
fun Point.origin() | Point {
    return Point{x = 0, y = 0};
}
```

Синтаксис: `Type.methodName`.

### Виклик статичних методів

Статичні методи викликаються через тип.

```avenir
var origin | Point = Point.origin();
```

Статичні методи не мають доступу до полів конкретного екземпляра.

## Видимість методів

Для методів діють ті самі правила видимості, що і для функцій:

- **Публічні методи**: з `pub`, доступні з інших модулів
- **Приватні методи**: без `pub`, доступні тільки всередині модуля

```avenir
pub struct Point {
    x | int
    y | int
}

pub fun (self | Point).distance() | float {
}

fun (self | Point).internal() | void {
}
```

## Перевантаження методів

Avenir не підтримує перевантаження методів. Ім'я методу повинно бути унікальним у межах типу.

## Методи і функції

Методи — це по суті функції з отримувачем.

Наведене нижче концептуально еквівалентне:

```avenir
fun (self | Point).sum() | int {
    return self.x + self.y;
}

fun point_sum(p | Point) | int {
    return p.x + p.y;
}
```

Методи дають природніший синтаксис для об'єктно-орієнтованого стилю.

## Вбудовані методи

Вбудовані методи для `list`, `string`, `bytes` і `dict` описані у `builtins.md`.
