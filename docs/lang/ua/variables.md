# Змінні

Змінні в Avenir мають оголошуватися явно із зазначенням типу.

## Оголошення змінної

Змінні оголошуються за допомогою `var`:

```avenir
var name | type = value;
```

Анотація типу обов'язкова. Початкове значення також обов'язкове.

Приклад:

```avenir
var x | int = 10;
var name | string = "Avenir";
var is_ready | bool = true;
```

## Присвоєння

Змінним можна присвоювати нові значення через оператор `=`:

```avenir
var x | int = 10;
x = 20;
```

Значення має бути сумісним із типом змінної.

## Область видимості

Змінні видимі в межах блоку, де вони оголошені:

```avenir
{
    var x | int = 10;
    // x is available here
}
// x is not available here
```

Параметри функції видимі всередині тіла функції.

## Видимість змінних

Змінні завжди приватні для своєї області видимості. Експортувати змінну з модуля не можна.

## Вимоги до типів

Усі змінні повинні мати явну анотацію типу. Виведення типів не підтримується.

Приклад:

```avenir
var x | int = 10;
var y = 10;
```
