# Оператори

Цей документ описує оператори Avenir і правила їх типізації.

## `+` (додавання та конкатенація рядків)

Оператор `+` розв'язується type checker-ом і підтримує рівно дві форми:

- числове додавання: `int + int`, `float + float` або змішане `int`/`float`
- конкатенація рядків: `string + string`

Неявні перетворення не виконуються. Зокрема:

- `string + int` — помилка компіляції
- `int + string` — помилка компіляції

Приклади:

```avenir
var a | string = "hello " + "world";
var n | int = 1 + 2;
var f | float = 1 + 2.5;
```

Якщо потрібно вбудувати не-рядкові значення, використовуйте інтерполяцію рядків:

```avenir
print("sum = ${1 + 2}");
```
